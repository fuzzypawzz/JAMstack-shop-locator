About

The Shop Locator is static web application made with the JAMStack architecture in mind. The only 'back-end' this app has, is an Azure Function serving as it's primary web API.

The JAMStack Shop Locator is created by Jannik Maag, as part of a school project in Cloud Hosting Technologies at CPH Business. - Copenhagen 2021.
License: MIT.

Made using

  • HTML
  • Sass SCSS
  • Microsoft Azure Functions
  • Handlebars.js
  • Google Maps JS API
